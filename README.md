# 🚀 Dilbas Performance Suite

Uma suíte de otimização para Windows focada em desenvolvedores.

## Objetivos

- Melhorar o desempenho durante programação.
- Automatizar tarefas repetitivas.
- Limpar caches e arquivos temporários.
- Facilitar o desenvolvimento Java, Python e outras linguagens.

## Tecnologias

- PowerShell
- JSON
- Git
